# 2. Quick Start Guide

## Installation

```bash
composer require wordpress/php-ai-client
```

## Basic Usage

```php
use WordPress\AiClient\AiClient;

// Simple text generation
$result = AiClient::generateTextResult('Write a haiku about coding');
echo $result->toText();

// With configuration
$result = AiClient::prompt('Write a product description')
    ->usingTemperature(0.7)
    ->generateTextResult();

// Image generation
$image = AiClient::generateImageResult('Mountain landscape');
echo $image->toImageFile()->getUrl();
```
## Provider Authentication

```php
use WordPress\AiClient\AiClient;

// Set up API keys
$registry = AiClient::defaultRegistry();
$registry->setProviderRequestAuthentication('openai', $_ENV['OPENAI_API_KEY']);
$registry->setProviderRequestAuthentication('anthropic', $_ENV['ANTHROPIC_API_KEY']);

// Use the SDK
$result = AiClient::prompt('Write a product description')->generateTextResult();
```

## WordPress Integration

```php
// WordPress plugin
add_action('init', function() {
    $content = AiClient::prompt('Generate blog content')
        ->generateTextResult()
        ->toText();
});
```

## Error Handling

```php
try {
    $result = AiClient::prompt('Generate content')->generateTextResult();
    echo $result->toText();
} catch (Exception $e) {
    error_log('AI generation failed: ' . $e->getMessage());
}
